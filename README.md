# 🎯 Interview Prep Hub

A curated, single-page resource hub designed to help software engineers **prepare smarter and faster** for technical interviews. Instead of hunting across the internet for quality material, this page brings together the most trusted, community-validated resources — all in one place.

---

## 🌐 Live Demo

Website: https://akapande-dev.github.io/InterviewPlaybook/

---

## 💡 Purpose

Technical interviews are hard — not because the problems are impossible, but because the preparation is scattered. Developers spend hours just figuring out *what* to study and *where* to study it.

**Interview Prep Hub** solves that. It is a personal reference page that centralises the highest-signal resources across every major interview category: data structures, JavaScript, React, system design, and AI-assisted coding. Whether you are preparing for your first job or targeting a senior/staff engineering role, this hub gives you a clear starting point and a direct path to each resource.

---

## 📦 What's Inside

### 🧩 1. DS & Algorithms — Coding Practice
**Source:** [Striver's 79 Last-Moment DSA Sheet – TakeUForward](https://takeuforward.org/dsa/strivers-79-last-moment-dsa-sheet-ace-interviews)

A battle-tested list of 79 carefully chosen problems that cover every major DSA topic — arrays, trees, graphs, dynamic programming, and more. Ideal for last-minute prep or structured daily practice. Used by thousands of engineers to crack top product companies.

---

### ⚡ 2. JavaScript Interview Questions
**Source:** [Top JavaScript Interview Questions – GreatFrontEnd (GitHub)](https://github.com/greatfrontend/top-javascript-interview-questions)

A community-driven, open-source collection of the most frequently asked JavaScript questions in frontend interviews. Covers core language concepts like closures, prototypes, event loops, async/await, and the `this` keyword — with clear explanations and code examples.

---

### ⚛️ 3. React Interview Questions
**Source:** [100 React Interview Questions – GreatFrontEnd Blog](https://www.greatfrontend.com/blog/100-react-interview-questions-straight-from-ex-interviewers)

100 React questions sourced directly from ex-interviewers at top tech companies. Topics span hooks, component lifecycle, state management, performance optimisation, reconciliation, and architectural patterns — exactly what interviewers test at senior levels.

---

### 🏗️ 4. System Design Interview Questions
**Source:** [64 System Design Questions – dev.to](https://dev.to/arslan_ah/64-system-design-interview-questions-ranked-from-easiest-to-hardest-260m)

64 system design questions ranked from easiest to hardest, making it easy to level up progressively. Covers URL shorteners, rate limiters, distributed caches, messaging queues, and large-scale architectures — essential for mid-to-senior engineering interviews.

---

### 🤖 5. Claude Code Interview Questions
**Source:** [YouTube – Claude Code Walkthrough](https://www.youtube.com/watch?v=MzhIr7BfpI0)

An emerging and increasingly relevant topic: how to use AI coding tools like Claude Code effectively during technical interviews and real-world engineering workflows. Video walkthroughs covering prompting strategies, agentic coding patterns, and practical use cases.
